# Hogar Contable

Aplicación de escritorio para finanzas personales hecha con **Wails v2 (Go) + React + TypeScript + TailwindCSS**.

## Stack

| Capa | Tecnología |
|------|-----------|
| Frontend | React 18 + TypeScript + Vite |
| Estilos | TailwindCSS + shadcn/ui + Recharts |
| Backend | Go 1.25 (Clean Architecture) |
| BD | SQLite (modernc.org/sqlite) |
| Nativo | Wails v2 (WebView2 / WebKit2GTK) |
